# User Stories
- As a user, I want to search for songs so that I can find music I like.
- As a user, I want to get recommendations based on a song so that I can discover new music.
- As a user, I want to have a playlist so that I can save my favorite songs.
- As a user, I want to get recommendations based on my favorite songs. 
- As a user, I want to add new songs. 
- As a user, I want to remove songs that I no longer want to see.
